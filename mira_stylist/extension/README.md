# MIRA Stylist Browser Extension Scaffold

This folder contains a minimal future-facing browser extension scaffold.

Intent:

- detect selected or prominent product images
- surface a "Try with MIRA Stylist" action
- send image-first payloads to the MIRA Stylist backend

Current status:

- stub manifest
- stub content script
- stub background worker

Not implemented:

- auth
- retailer-specific parsers
- production messaging and telemetry
- polished UI
